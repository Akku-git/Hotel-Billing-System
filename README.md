# Hotel Billing System

## 📌 Description
The Hotel Billing System is a web-based application designed to manage restaurant operations efficiently. It allows users to create orders, generate bills, manage menu items, track expenses, and view sales reports.

## 🚀 Features
- Order management (Dine-in & Pickup)
- Bill generation with print support
- Menu management (Add/Edit items)
- Expense tracking system
- Sales reports with graphical charts
- Real-time order updates

## 🛠️ Technologies Used
- HTML, CSS (UI Design)
- JavaScript (Logic & Functionality)
- Tailwind CSS (Styling)
- Chart.js (Reports & Graphs)

## 🎯 Objective
To simplify restaurant billing and management by providing an easy-to-use digital system for handling orders, expenses, and financial reports.
